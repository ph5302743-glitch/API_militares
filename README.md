# API REST — Controle de Militares

Esta API foi desenvolvida como parte da atividade prática do curso Fullstack (Turma 183) do Senac, com o objetivo de aplicar na prática os conceitos de back-end aprendidos em sala: construção de uma API REST utilizando Node.js, Express e PostgreSQL, seguindo a arquitetura MVC (Model-View-Controller).

O sistema permite o gerenciamento completo de militares, oferecendo operações de cadastro, listagem, atualização (total e parcial) e exclusão de registros, conectando-se a um banco de dados relacional PostgreSQL para persistência das informações.

Este projeto teve como base a arquitetura de um projeto anterior de cadastro de alunos desenvolvido em sala, reaproveitando a estrutura de pastas e os padrões de organização do código.
